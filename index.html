How to use this dashboard

Start wide, narrow the time window, then dig into specific titles.

  1. Use the timeline brush to set the year range you want to analyze.
  2. Scan the color matrix to spot region/genre clusters and click to filter.
  3. Open the deep dive to compare budgets vs. ratings and pick titles.
  4. Review the global overview for summary stats and detailed title lookups.
